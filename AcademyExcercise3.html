입력

없음

호출

nowTime();

출력

현재 시간 : 10시 30분 25초

코드 참조

import java.util.Date;

class Example 
{
    public static void main(String[] args) 
    {
        Date now = new Date();
        System.out.printf("%d시 %d분 %d초\n", now.getHours(), now.getMinutes(), now.getSeconds());
    }
}

메소드 서명

  • void nowTime()

추가 조건

  • 현재 시간을 오전/오후로 출력하시오.
    • void nowTime_AMPM()
    • 14시 30분 25초 → 오후 2시 30분 25초

수업

  • 사용자로부터 무언가를 입력받기(BufferedReader)
  • 문자열 형식 출력하기

메소드 서명

  • 흐린 날 사과 나무의 성잘률 : 2㎝
