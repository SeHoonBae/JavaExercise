배열(1차원)

요구사항.001

숫자를 5개 입력받아 배열에 담은 뒤 역순으로 출력하시오.

입력

숫자 : 5
숫자 : 1
숫자 : 7
숫자 : 2
숫자 : 3

출력

nums[4] = 3
nums[3] = 2
nums[2] = 7
nums[1] = 1
nums[0] = 5

요구사항.002

학생의 이름을 N개 입력받아 아래와 같이 출력하시오.

입력

학생 수 : 6
학생명 : 홍길동
학생명 : 아무개
학생명 : 하하하
학생명 : 호호호
학생명 : 후후후
학생명 : 헤헤헤

출력

입력한 학생은 총 6명입니다.
1. 헤헤헤
2. 후후후
3. 호호호
4. 하하하
5. 아무개
6. 홍길동

추가조건

  • 입력받은 학생 이름은 일단 동적 배열에 저장한다.
  • 입력받은 순서의 반대로 출력한다.

요구사항.003

1~20 사이의 난수를 담고 있는 배열을 생성하고 최대값과 최소값을 출력하시오.

입력

없음

출력

원본 : 10, 9, 3, 18, 15, 9, 5, 4, 3, 11, 15, 18, 9, 14, 18, 5, 8, 7, 9, 17
최대값 : 18
최소값 : 3

조건

  • 난수를 20개 발생 후 배열에 넣는다.
  • 난수는 1 ~ 20 사이

요구사항.004

난수를 담고 있는 배열을 생성한 뒤 아래와 같이 출력하시오.

입력

최대 범위 : 15
최소 범위 : 5

출력

원본 : 10, 2, 3, 20, 15, 9, 5, 1, 3, 11, 15, 18, 9, 14, 18, 5, 8, 2, 1, 17
결과 : 10, 15, 9, 5, 11, 15, 9, 14, 5, 8

조건

  • 난수를 20개 발생 후 배열에 넣는다.
  • 난수는 1 ~ 20 사이
  • 배열을 탐색하여 범위에 만족하는 숫자만 출력한다.

요구사항.005

중복되지 않는 임의의 숫자 5개를 만드시오.

입력

없음

출력

[5, 6, 1, 3, 2]

조건

  • 숫자의 범위 : 1 ~ 9

추가조건

  • 숫자의 범위를 입력받으시오.
  • 숫자의 개수를 입력받으시오.

요구사항.006

배열에 요소를 삽입하시오.

입력

삽입 위치 : 2
값 : 100

출력

원본 : [5, 6, 1, 3, 2, 0, 0, 0, 0, 0]
결과 : [5, 6, 100, 1, 3, 2, 0, 0, 0, 0]

조건

  • 배열의 시작 길이 : 10
  • 난수 대입

요구사항.007

배열에 요소를 삭제하시오.

입력

삭제 위치 : 2

출력

원본 : [5, 6, 1, 3, 2]
결과 : [5, 6, 3, 2, 0]

조건

  • 배열의 시작 길이 : 5
  • 난수 대입
